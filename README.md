# Historical Stock Price Analysis (1970–2020)

## Project Overview

This project analyzes historical stock price data from 1970 to 2020. The goal is to explore long-term trends in stock prices and trading volume, and compare market behaviour across different decades.

The analysis includes data cleaning, merging datasets, statistical analysis, and visualizations such as time series plots, histograms, and box plots.

---

## Dataset

The project uses two datasets:
- historical_stock_prices.csv (daily stock prices)
- historical_stocks.csv (company information such as sector and industry)

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

---

## Key Steps

1. Data loading and inspection
2. Data cleaning (missing values and duplicates)
3. Merging datasets
4. Creating decade-based features
5. Exploratory data analysis
6. Visualization of trends across decades
7. Statistical summary and comparison

---

## Key Findings

- Stock prices generally increased over time.
- Trading volume increased significantly in later decades.
- Later decades show higher volatility compared to earlier decades.
- The stock market has grown in size and complexity over time.
