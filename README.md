# Historical Stock Market Analysis (1970–2020)

## Project Overview

This project analyzes historical stock market data to identify trends in stock prices, trading volume, and market behavior across different decades.

The goal of this analysis was to use exploratory data analysis (EDA) techniques to understand how stock market activity has changed over time.

---

## Dataset

The project uses two datasets:

**Historical Stock Information**

* Company ticker
* Exchange
* Company name
* Sector
* Industry

**Historical Stock Prices**

* Open price
* Close price
* High and low prices
* Trading volume
* Adjusted close price
* Date

The datasets were combined using the stock ticker to include both company information and historical price data.

---

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Google Colab

---

## Analysis Performed

The project includes:

* Data cleaning and preparation
* Missing value and duplicate handling
* Date conversion and time series preparation
* Feature engineering using Year and Decade categories
* Dataset merging
* Summary statistics
* Stock price and volume visualizations

---

## Key Findings

The analysis showed:

* Stock prices generally increased over time.
* Trading volume grew significantly across decades.
* Later decades showed greater market activity and price variation.
* Some periods contained larger price fluctuations and outliers.

---

## Conclusion

This project demonstrates how exploratory data analysis can be used to identify long-term patterns in financial data. By analyzing historical stock prices and trading activity, the project highlights changes in market growth, participation, and volatility over time.
